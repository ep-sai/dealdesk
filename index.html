CSM Leistungsübersicht

Einmalige Einrichtung – danach funktioniert die App wie normale Software für dein ganzes Team.

Schritt 1 – Supabase Projekt erstellen Gehe zu supabase.com → "Start for free" → Neues Projekt erstellen. Wähle Region eu-central-1 (Frankfurt).
Schritt 2 – Tabelle anlegen Im Supabase Dashboard: SQL Editor → "New query" → folgenden SQL-Code einfügen und ausführen:
create table kunden (
  id uuid default gen_random_uuid() primary key,
  created_at timestamptz default now(),
  kunde text not null,
  angebotsnr text,
  angebotsdatum text,
  vertragsbeginn text,
  laufzeit text,
  laufzeit_monate int,
  verlaengerung text,
  vertragsende_iso text,
  invest_pa numeric,
  invest_label text,
  leistungen jsonb,
  notiz text
);
alter table kunden enable row level security;
create policy "allow_all" on kunden for all using (true) with check (true);
Schritt 3 – API-Keys einfügen Im Supabase Dashboard: Project Settings → API → URL und anon key kopieren.
Wird lokal gespeichert – kein Kollege sieht ihn.
